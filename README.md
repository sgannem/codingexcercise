### how to run the project

#### pre-requisites
* jdk 17
* maven 3.0.5 and above
#### how to run project
```
git clone <project-git-hub-url>
cd <cloned-project>
mvn clean install
java -jar target/codeassignment-0.0.1-SNAPSHOT.jar
```
